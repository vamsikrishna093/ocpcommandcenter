#                               Resource, BatchSpanProcessor, etc.
#
# opentelemetry-instrumentation-fastapi
#                             — auto-instruments every FastAPI request:
#                               creates a SERVER span, fills in
#                               http.method / http.route / http.status_code.

#
# opentelemetry-exporter-otlp-proto-grpc
#                             — sends spans/metrics to an OTel Collector
#                               (or Tempo / Prometheus) over gRPC (default).
#
# opentelemetry-exporter-otlp-proto-http
#                             — alternative HTTP/protobuf exporter.
#                               Useful if gRPC is blocked in your network.
#
# grpcio                      — gRPC runtime required by the gRPC exporter.
